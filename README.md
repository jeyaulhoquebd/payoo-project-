# Payoo MFS - A Modern Mobile Financial Service Web App 📱💸

Payoo is a fully responsive Mobile Financial Service (MFS) web application demo inspired by popular digital wallets like bKash and Nagad. This project focuses on providing a clean, intuitive, and interactive user interface for digital transactions.

## ✨ Features
This project includes essential MFS functionalities with a focus on UI/UX:

* **Interactive Dashboard:** A sleek and modern home screen after secure login.
* **Balance Privacy:** An interactive toggle feature to show or hide the account balance.
* **Add Money:** Functional UI to simulate adding funds from banks or cards.
* **Cash Out:** Simplified process for withdrawing funds using agent numbers.
* **Transfer Money:** Instant peer-to-peer money transfer simulation.
* **Pay Bill:** Interface for paying utility and other service bills.
* **Transaction History:** A dedicated section to keep track of all recent activities.
* **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop devices using a mobile-first approach.

## 🛠️ Technologies Used
- **HTML5:** For semantic web structure.
- **Tailwind CSS:** For modern, utility-first styling and responsive layouts.
- **JavaScript (ES6+):** For handling DOM manipulation, UI toggles, and application logic.
- **Vercel:** For fast and reliable hosting.
