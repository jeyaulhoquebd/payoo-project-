# 📱 Payoo MFS - A Modern Digital Wallet Experience

Payoo is a sleek, fully responsive Mobile Financial Service (MFS) web application demo. Inspired by leading digital wallets like bKash and Nagad, it bridges the gap between clean UI design and functional financial interactions.

---

## 🚀 Overview
This project focuses on delivering a high-quality user experience for digital transactions. From managing balances to simulating bill payments, Payoo provides a comprehensive look at how modern fintech interfaces operate.

## ✨ Core Features
- 🏠 **Interactive Dashboard** – A clean and modern landing page post-login.
- 👁️ **Balance Toggle** – Securely show or hide account balance with a single click.
- 💸 **Financial Modules** – Dedicated interfaces for:
    - **Add Money:** Simulate bank-to-wallet transfers.
    - **Cash Out:** Withdraw funds via agent numbers.
    - **Transfer Money:** Instant peer-to-peer simulation.
    - **Pay Bill:** Seamless utility bill payment UI.
- 📜 **Transaction History** – Track every movement with a detailed history log.
- 📱 **Mobile-First Design** – Fully optimized for mobile users while staying perfectly responsive on desktop and tablets.
